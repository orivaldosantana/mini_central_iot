# Sistema Embarcado

## LED

Pinos:

| LED | ESP32   |
| --- | ------- |
| -   | GND     |
| +   | GPIO 02 |

## Sensor de temperatura e umidade do ar

Pinos:

| Sensor DS18B20 | ESP32 |
| -------------- | ----- |
| -              | GND   |
| +              | 3.3v  |
| S              | D14   |

## Sensor LDR

Pinos:

| Sensor de LDR | ESP32 |
| ------------- | ----- |
| -             | GND   |
| +             | 3.3v  |
| S             | D33   |
